AllMyBooksArePacked

Programming and data analysis challenge
Programmed in Python

**Download all files in AllMyBooksArePacked folder and build main.py to run program

main.py 	-> main program control
parser.py 	-> HTML parser and data extractor
item.py		-> item module, books
container.py	-> container module used in sorting
boxsort.py	-> sort logic (linear and dynamic)
export.py	-> JSON file export module
test.py		-> non-comprehensive test file suite
EXTENSIONS.txt	-> notes on additional problems
bookspacked.txt	-> JSON file exported from main program
